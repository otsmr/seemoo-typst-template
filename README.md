# (unofficial) SEEMOO Thesis Template

A [Typst](https://typst.app/) template for SEEMOO thesis and papers in the department of Computer Sciences.

The original LaTeX template can be found [here](https://git.rwth-aachen.de/seemoo-lab/staff/templates/seemoo-thesis-template).
